Projetos

  • Biblioteca: Sistema desktop desenvolvido em Java, utilizando Swing para a interface gráfica. O projeto foi criado para apoiar o gerenciamento de uma distribuidora de bebidas, oferecendo funcionalidades como sistema de login, cadastros de clientes, produtos, usuários, fornecedores e funcionários, além de menus de navegação intuitivos. Para a persistência e manipulação dos dados, foi implementada a integração direta com o banco de dados MySQL.

  • Auto Solar: Desenvolvimento de um protótipo funcional de estação de carregamento de automóveis elétricos, utilizando energia solar fotovoltaica. O projeto envolveu planejamento e modelagem 3D no Fusion 360, impressão 3D em PLA, montagem de estrutura com totens de carregamento, painel solar e regulador de tensão conectado a uma bateria de armazenamento. O protótipo permite carregar dispositivos eletrônicos, como celulares, demonstrando a viabilidade do conceito. O projeto enfatiza sustentabilidade, inovação e mobilidade urbana, contribuindo para a redução da pegada de carbono e promovendo o uso de fontes de energia limpa e renovável.
