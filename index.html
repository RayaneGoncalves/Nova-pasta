Projetos

Biblioteca

Sistema desenvolvido em Java com interface gráfica em Swing. Possui funcionalidades como login, cadastro de clientes, produtos e usuários. Os dados são armazenados e manipulados em um banco de dados MySQL integrado ao sistema.

Auto Solar

Protótipo de uma estação de carregamento para carros elétricos, movida a energia solar. O projeto incluiu modelagem 3D no Fusion 360, impressão 3D em PLA, montagem de estrutura com totens de carregamento, painel solar, regulador de tensão e bateria de armazenamento. O protótipo permite carregar dispositivos como celulares.
